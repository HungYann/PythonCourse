# Python编程与人工智能

用户的目标是：精通Python， 于此同时，能够在学术论文 教学与会议中使用。





### 课程方案



### Python 编程与学术研究应用课程大纲

**课程代码**：PY-7001 ｜ **学分**：4 ｜ **课时**：40学时 ｜ **先修要求**：无





### 课程学习成果

学完本课程后，学生能够：

1. 掌握 Python 核心语法与编程范式，独立构建可运行程序
2. 运用 Python 进行数据处理、统计分析与科学计算
3. 使用 Python 工具链支持学术论文的数据分析与可视化
4. 规范输出学术级图表、表格及可复现的研究代码 (LaTeX 表格生成、参考文献管理、论文图表导出)
