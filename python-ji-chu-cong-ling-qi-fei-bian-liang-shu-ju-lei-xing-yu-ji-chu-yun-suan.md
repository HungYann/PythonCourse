### 📦 变量（8 分钟）

#### 什么是变量？

变量就像是一个**贴了标签的盒子**。盒子里可以放任何东西，标签就是变量名。

```
┌─────────────┐
│   "小明"    │  ← 里面装的值
└─────────────┘
      ↑
    name       ← 标签（变量名）
```

#### 定义变量

Python 中定义变量极其简单，直接写 `变量名 = 值`：

```python
name = "小明"        # 定义一个变量，存储名字
age = 18             # 存储年龄
height = 1.75        # 存储身高
is_student = True    # 存储是否是学生
```

#### 使用变量

```python
name = "小明"
age = 18

print(name)          # 输出：小明
print(age)           # 输出：18
print("我叫" + name + "，今年" + str(age) + "岁")
# 输出：我叫小明，今年18岁
```

#### 变量命名规则 📏

| 规则                | 例子                        |
| ----------------- | ------------------------- |
| ✅ 只能用字母、数字、下划线    | `my_name`, `score1`       |
| ✅ 不能以数字开头         | `name1` ✅，`1name` ❌       |
| ✅ 区分大小写           | `Name` 和 `name` 是两个不同变量   |
| ✅ 不能使用 Python 关键字 | `if`, `for`, `while` 等不可用 |

> 🏆 **命名小技巧：** 用有意义的名字！`student_score` 比 `ss` 好得多。

#### 修改变量

变量的值可以随时更改：

```python
score = 60
print(score)   # 输出：60

score = 95
print(score)   # 输出：95
```

### 🧱 数据类型（12 分钟）

Python 有四种最基本的数据类型，分别对应生活中不同种类的信息。

#### 总览

```python
#  类型         示例值              用途
#  整数 int     42, -10, 0         年龄、数量、排名
#  浮点数 float 3.14, -0.5, 1.0   身高、价格、成绩
#  字符串 str   "hello", '世界'    文字、名字、句子
#  布尔值 bool  True, False        是/否、开/关
```

***

#### 1. 整数 `int`

没有小数点的数字，包括负数和零。

```python
year = 2024
temperature = -5
count = 0

print(type(year))   # 输出：<class 'int'>
```

***

#### 2. 浮点数 `float`

带小数点的数字（来自"浮点"概念）。

```python
price = 9.99
pi = 3.14159
weight = 65.5

print(type(price))  # 输出：<class 'float'>
```

> ⚠️ **常见陷阱：**
>
> ```python
> print(0.1 + 0.2)   # 输出：0.30000000000000004（不是 0.3！）
> ```
>
> 这是计算机浮点存储的特性，初学者先了解即可，不必担心。

***

#### 3. 字符串 `str`

由引号包裹的文字内容，单引号双引号均可。

```python
greeting = "你好"
city = '北京'
sentence = "Python 很有趣！"

print(type(greeting))  # 输出：<class 'str'>
```

**字符串的常用操作**

```python
name = "Python"

# 拼接（用 + 号）
print("我爱" + name)          # 输出：我爱Python

# 重复（用 * 号）
print("哈" * 3)               # 输出：哈哈哈

# 获取长度
print(len(name))              # 输出：6

# 转大/小写
print(name.upper())           # 输出：PYTHON
print(name.lower())           # 输出：python

# 索引（从 0 开始）
print(name[0])                # 输出：P
print(name[-1])               # 输出：n（最后一个）
```

**f-string：优雅的字符串格式化**

```python
name = "小明"
age = 18

# 旧方式（繁琐）
print("我叫" + name + "，今年" + str(age) + "岁")

# f-string（推荐）
print(f"我叫{name}，今年{age}岁")   # 输出：我叫小明，今年18岁
```

> 💡 在引号前加 `f`，就可以用 `{}` 直接嵌入变量，非常方便！

***

#### 4. 布尔值 `bool`

只有两个值：`True`（真）和 `False`（假），注意首字母大写。

```python
is_raining = False
has_homework = True

print(type(is_raining))   # 输出：<class 'bool'>
```

布尔值常用于条件判断：

```python
age = 20
is_adult = age >= 18
print(is_adult)           # 输出：True
```

***

#### 类型转换

不同类型之间可以相互转换：

```python
# 字符串 → 整数
num_str = "42"
num = int(num_str)
print(num + 1)        # 输出：43

# 整数 → 字符串
age = 18
age_str = str(age)
print("今年" + age_str + "岁")  # 输出：今年18岁

# 整数 → 浮点数
x = float(5)
print(x)              # 输出：5.0

# 查看类型
print(type(num))      # 输出：<class 'int'>
```

**类型转换速查表**

| 转换目标 | 函数        | 示例                       |
| ---- | --------- | ------------------------ |
| 整数   | `int()`   | `int("10")` → `10`       |
| 浮点数  | `float()` | `float("3.14")` → `3.14` |
| 字符串  | `str()`   | `str(42)` → `"42"`       |
| 布尔值  | `bool()`  | `bool(0)` → `False`      |

### ➕ 基础运算（10 分钟）

#### 1. 算术运算符

```python
a = 10
b = 3

print(a + b)    # 加法：13
print(a - b)    # 减法：7
print(a * b)    # 乘法：30
print(a / b)    # 除法：3.3333...（结果是浮点数）
print(a // b)   # 整除：3（舍去小数）
print(a % b)    # 取余：1（10 除以 3 余 1）
print(a ** b)   # 幂运算：1000（10 的 3 次方）
```

**运算优先级（和数学一样）**

```
() 括号  →  ** 幂  →  * / // %  →  + -
```

```python
print(2 + 3 * 4)      # 输出：14（先乘后加）
print((2 + 3) * 4)    # 输出：20（括号优先）
```

***

#### 2. 比较运算符

比较运算符的结果总是布尔值（True 或 False）：

```python
a = 10
b = 5

print(a > b)    # 大于：True
print(a < b)    # 小于：False
print(a >= 10)  # 大于等于：True
print(a <= 9)   # 小于等于：False
print(a == b)   # 等于：False
print(a != b)   # 不等于：True
```

> ⚠️ **注意：** `=` 是赋值，`==` 才是比较是否相等！

***

#### 3. 赋值运算符

```python
x = 10

x += 5    # 等同于 x = x + 5，x 现在是 15
x -= 3    # 等同于 x = x - 3，x 现在是 12
x *= 2    # 等同于 x = x * 2，x 现在是 24
x /= 4    # 等同于 x = x / 4，x 现在是 6.0
x //= 2   # 等同于 x = x // 2，x 现在是 3.0
x **= 3   # 等同于 x = x ** 3，x 现在是 27.0

print(x)  # 输出：27.0
```

***

#### 4. 字符串运算（回顾）

```python
# 字符串也支持部分运算符
first = "Hello"
second = " World"

print(first + second)     # 拼接：Hello World
print(first * 3)          # 重复：HelloHelloHello
print(len(first + second))# 长度：11
```

### 🏁 综合练习（5 分钟）

现在把今天学的所有内容组合起来，写一个小程序！

#### 练习：个人信息计算器

```python
# =============================
# 个人信息计算器
# =============================

# 接收用户输入
name = input("请输入你的名字：")
birth_year = int(input("请输入你的出生年份："))
height_cm = float(input("请输入你的身高（厘米）："))

# 计算年龄和身高（米）
current_year = 2024
age = current_year - birth_year
height_m = height_cm / 100
is_adult = age >= 18

# 输出结果
print("=" * 30)
print(f"姓名：{name}")
print(f"年龄：{age} 岁")
print(f"身高：{height_m:.2f} 米")    # :.2f 表示保留两位小数
print(f"是否成年：{is_adult}")
print("=" * 30)
```

**示例运行结果：**

```
请输入你的名字：小明
请输入你的出生年份：2006
请输入你的身高（厘米）：175
==============================
姓名：小明
年龄：18 岁
身高：1.75 米
是否成年：True
==============================
```

### 📝 本课知识总结

```
变量
├── 定义：变量名 = 值
├── 命名：字母/数字/下划线，不以数字开头
└── 修改：重新赋值即可

数据类型
├── int（整数）：42, -10, 0
├── float（浮点数）：3.14, -0.5
├── str（字符串）："hello", '世界'
└── bool（布尔值）：True, False

类型转换
├── int() / float() / str() / bool()
└── type() 查看类型

运算符
├── 算术：+ - * / // % **
├── 比较：> < >= <= == !=
└── 赋值：+= -= *= /= //= **=
```
