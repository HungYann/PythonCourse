# Python基础（从零起飞） - 环境搭建：本地 + 云端双方案

**一、教学目标**

* 掌握 Python 基础概念与运行机制，理解其作为解释型语言的特点
* 理解解释型语言（如 Python）与编译型语言（如 Golang）的执行原理差异
* 能够独立完成 Python 本地开发环境搭建并运行程序
* 掌握云端 Python 环境（如 Google Colab）的基本使用
* 掌握 Git 与 GitHub 的基础使用方法
* 理解代码管理、版本迭代与协同开发流程

**二、教学内容**&#x20;

**1. Python 基础与运行机制**

* Python 语言简介与应用场景
* 解释型语言执行过程
* 编译型语言（如 Golang、Java）运行机制对比

**2. 本地开发环境搭建**

* Python 安装与环境变量配置
* 命令行运行 Python 程序
* 基本开发工具（VS Code / Terminal）

**3. 云端开发环境实践**

* Google Colab 注册与使用
* 浏览器中编写与运行 Python
* 本地与云端环境对比

**4. 版本控制与代码管理**

* Git 基本概念（仓库、提交、分支）
* 常用命令（init / add / commit / push）
* GitHub 使用（创建仓库、上传代码）

**5. 协同开发基础**

* 代码更新流程（提交、合并）
* 多人协作与冲突解决入门
* 开源参与流程（Fork / Pull Request）

**三、能力达成**

* 独立搭建 Python 开发环境（本地 + 云端）
* 编写并运行基础 Python 程序
* 使用 Git 与 GitHub 管理代码
* 完成基础团队协作开发流程
